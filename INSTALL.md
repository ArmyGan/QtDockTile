### init repository (from git)

$ git clone git://github.com/gorthauer/QtDockTile.git <br />
$ cd QtDockTile <br />
$ git submodule update --init <br />

### build

$ mkdir build <br />
$ cmake .. <br />
$ make <br /> 

### install

$ sudo make install <br />

### example 
Just launch simpledock or simpledock.app on os x

### uninstall

$ sudo make uninstall
